# AI Recruitment Chatbot System

## Overview
A full-stack AI-powered recruitment platform designed to automate candidate screening and simulate real interview environments. The system combines conversational AI with behavioural monitoring to improve hiring efficiency and integrity.

This project was developed to replicate real-world online assessment systems used by companies, integrating chatbot interaction, activity tracking, and backend session management.

---

## Key Features

### 🤖 AI Chatbot Interaction
- Automated interview conversation flow
- Dynamic question handling using AI logic
- Structured candidate response collection

### 🛡️ Candidate Monitoring System
- Copy-paste detection during assessments
- Tab switching / focus tracking
- Behavioural activity logging
- Face detection support using OpenCV (Haar Cascade)

### 🎥 Interview Control & Tracking
- Session-based interview management
- Candidate data and response storage
- Real-time backend session tracking

### 🔗 Backend System
- RESTful architecture using ASP.NET Core
- Service-based modular design
- Integration with Python-based components

---

## 🧱 Tech Stack

**Backend**
- ASP.NET Core MVC
- C#
- Entity Framework (or custom data handling)

**Frontend**
- HTML, CSS, JavaScript
- Bootstrap (via CDN)

**AI / ML Components**
- Chatbot logic (LLM / rule-based hybrid)
- OpenCV Haar Cascade (face detection)
- Python integration (`scraper.py`)

**Database**
- SQL Server / Local DB

---

## 🏗️ System Architecture

- **Controllers** → Handle API requests and routing  
- **Services** → Business logic (Chat, Notifications, Session handling)  
- **Models** → Data structures for interview and user data  
- **Views** → UI for chatbot and admin interface  
- **Python Layer** → Data scraping / external processing
